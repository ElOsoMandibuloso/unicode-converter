## 🇬🇧 English

A fast, offline, and real-time text encoder and decoder. Convert instantly between Text, Hex, Binary, Base64, Unicode, and many other data formats directly in your browser.

### Key Features
- **Real-Time Conversion:** Type or paste data into any field and see all other formats update instantly.
- **14+ Supported Formats:** Text, Decimal, Binary, Octal, Hexadecimal, Unicode, Base64, URL Encode, HTML Entities, ROT13, Base32, Base58, Morse Code, and Punycode.
- **Fully Offline:** All conversions are computed client-side using JavaScript. No APIs, no tracking, and no rate limits.
- **Smart Text Handling:** Supports multi-line input, line break toggling, output limiting, and smart alternative character fallback (e.g., stripping accents for Morse code).
